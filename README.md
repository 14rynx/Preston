# Preston

Preston is a Python library for accessing EVE Online's ESI API.

## Installation

From [pip](https://pip.pypa.io/en/stable/):

```bash
pip install preston
```

From GitHub:

```bash
git clone https://github.com/Celeo/Preston.git
cd Preston
python setup.py install
```

## Initialization

```python
from preston import Preston

preston = Preston()
```
